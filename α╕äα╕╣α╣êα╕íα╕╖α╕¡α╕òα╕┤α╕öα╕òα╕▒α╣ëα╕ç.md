# คู่มือติดตั้ง SMART Nano Learning (ฉบับสมบูรณ์)

ชุดนี้แก้ปัญหา CORS ด้วยเทคนิค JSONP และทุกหน้าเป็นแบบ standalone (CSS + โค้ดอยู่ในไฟล์เดียว ไม่พึ่งไฟล์ภายนอก จึงไม่มีปัญหา cache ค้าง)

## ไฟล์ในชุด (8 ไฟล์)

| ไฟล์ | ใช้ที่ไหน | หน้าที่ |
|------|----------|---------|
| Code.gs | Apps Script | Backend API (JSONP) |
| index.html | GitHub | หน้าแรก + เข้าสู่ระบบ |
| lessons.html | GitHub | บทเรียน 6 ชุด |
| quiz.html | GitHub | ทำ Quiz |
| badges.html | GitHub | Badge 12 อัน |
| leaderboard.html | GitHub | ตารางอันดับ |
| checkin.html | GitHub | เช็คอิน QR |
| chatbot.html | GitHub | AI ครูประกันภัย |

**หมายเหตุ:** ไม่มี config.js, api.js, nav.js, style.css อีกต่อไป เพราะยุบเข้าไปในทุกไฟล์แล้ว

## ส่วนที่ 1: Apps Script (ใช้ของเดิมได้ ถ้า Code.gs เป็นเวอร์ชัน JSONP แล้ว)

ถ้ายังไม่ได้อัปเดต Code.gs เป็นเวอร์ชัน JSONP:
1. เปิด Google Sheets → Extensions → Apps Script
2. ลบ Code.gs เดิม วาง Code.gs ใหม่จากชุดนี้
3. Deploy → Manage deployments → ✏️ Edit → Version: New version → Deploy
4. **URL เดิมไม่เปลี่ยน** ไม่ต้องแก้อะไรใน HTML

**สำคัญ:** URL ของ Apps Script ถูกฝังอยู่ในทุกไฟล์ HTML แล้ว (บรรทัด `var API_URL = ...`)
ถ้า URL เปลี่ยน ต้องแก้ในทุกไฟล์ HTML — แต่ถ้าใช้ Manage deployments → Edit URL จะไม่เปลี่ยน

---

## ส่วนที่ 2: สร้าง GitHub Repo ใหม่ (แนะนำ)

1. สร้าง repository ใหม่ (Public)
2. Add file → Upload files
3. ลากไฟล์ HTML ทั้ง 7 ไฟล์เข้าไป (ไม่ต้องเอา Code.gs ขึ้น เพราะอยู่ใน Apps Script แล้ว)
4. Commit changes
5. Settings → Pages → Deploy from a branch → main → /(root) → Save
6. รอ 1-2 นาที ได้ URL เว็บ

---

## ส่วนที่ 3: ทดสอบ

1. เปิด URL เว็บใน Incognito (Ctrl+Shift+N)
2. ใต้ปุ่มเข้าสู่ระบบจะขึ้น "✅ พร้อมใช้งาน (24 คน)"
3. เลือกชื่อ → เข้าสู่ระบบ → เห็นคะแนน
4. กดเข้าสู่บทเรียน → ไม่เด้งวน → เห็นบทเรียน 6 ชุด
5. ทำ Quiz → ส่ง → เห็นคะแนน → เช็คใน Google Sheets

## ทำไมชุดนี้ถึงแก้ปัญหาได้

**ปัญหาเดิม:** fetch() ถูก CORS บล็อก เพราะ Apps Script redirect ข้ามโดเมน

**วิธีแก้:** เปลี่ยนเป็น JSONP — โหลดข้อมูลผ่าน `<script>` tag ซึ่งเบราว์เซอร์ไม่บล็อก CORS โดยธรรมชาติ

**ปัญหา cache:** เดิมแก้ api.js แล้วเบราว์เซอร์ยังใช้ตัวเก่า

**วิธีแก้:** ยุบทุกอย่างเข้าไฟล์ HTML แต่ละหน้า ไม่มีไฟล์ .js แยกให้ค้าง cache
